정규식(Regualar Expression)

    형식: /~~~/플래그(플러그인), /[~~~]/플래그(플러그인)                                         자바:^[]*$
    플래그(플러그인) :
      g(global) : 전체범위를 작업대상
        예) str.match(/a/) : 최초 1회 검색만 수행
            str.match(/a/g) : 모든 문자열 검색 수행
      s(single line) : 한줄의 걸쳐서 명령처리(text 필드) -   \n도 포함해서 처리
      m(multi line) : 여러줄의 내용에서의 명령처리(textarea 필드)
      i : 영문자일경우 대/소문자 구분없이 처리
      u : 유니코드처리시...

      메서드 : 
        ("문자열").match(/정규식/플래그)
        ("문자열").replace(/정규식/, "대체문자열")
        ("문자열").split(/정규식/)
        (/정규식/).test("문자열")
        (/정규식/).exec("문자열")
